# OOP 遊戲開發工具合集

這是我在大三開發 OOP 遊戲時所開發的綜合性圖片處理工具集，包含多種實用的圖片處理功能，使用 Python 和 Web 技術開發。這些工具主要用於遊戲開發過程中的圖片資源處理、UI 設計和遊戲素材製作。

## 🛠️ 工具列表

### 1. 區域標記工具 (`area_marker/`)
- **功能**：在圖片上標記特定區域並導出坐標數據
- **用途**：標記遊戲中的碰撞區域、可放置區域等
- **特點**：
  - 導入任意圖片作為標記基礎
  - 以圖片中心為原點(0,0)的坐標系
  - 使用兩點點擊方式繪製矩形區域（自動排序為左上到右下）
  - 支持移動已繪製的矩形
  - 支持 Ctrl+Z 撤銷操作
  - 切換標記區域類型（不能放的/水路）
  - 實時顯示鼠標位置的世界坐標
  - 刪除最後一個標記的區域
  - 清除所有標記區域
  - 導出標記數據為兩種不同格式的文本文件
  - 支持快速複製 Raw 數據到剪貼板
  - 自動記錄操作日誌

### 2. 圖片濾鏡工具 (`color_filter/`)
- **功能**：為圖片添加不同顏色的濾鏡效果
- **用途**：為遊戲素材添加色彩效果和氛圍
- **特點**：
  - 支持多種常見圖片格式（PNG、JPG、JPEG、BMP、GIF等）
  - 提供9種不同顏色的濾鏡：紅色、綠色、藍色、黃色、紫色、青色、橙色、棕色、粉色
  - 可調節濾鏡強度（0-100%）
  - 實時預覽濾鏡效果
  - 保留透明度（對於PNG圖片）
  - 處理後的圖片自動保存到原目錄
  - 處理後的圖片自動添加顏色標識到文件名

### 3. Windows圖標生成器 (`ico_generator/`)
- **功能**：將圖片轉換為Windows圖標（.ico）文件
- **用途**：為遊戲生成應用程序圖標
- **特點**：
  - 支持多種常用圖片格式轉換為ICO
  - 自動生成多種尺寸的圖標（16x16 到 256x256）
  - 保持透明度（如果源圖片支持）
  - 使用高質量的重採樣算法

### 4. 圖片裁剪與旋轉工具 (`image_crop_rotate/`)
- **功能**：網頁版圖片裁剪和旋轉工具
- **用途**：精確裁剪和旋轉遊戲素材
- **特點**：
  - 上傳任意圖片，自動調整至1024x1024的畫布上顯示
  - 使用點選頂點方式創建多邊形選區
  - 自動截取選定區域，並保持背景透明
  - 支援精確角度旋轉（1-359度）透過滑動條控制
  - 支援快速向左或向右旋轉90度
  - 輸出PNG格式圖片（支援透明背景）

### 5. 圖片批量處理工具 (`image_processor_main/`)
- **功能**：批量旋轉、縮放、重命名和背景填充圖片
- **用途**：遊戲素材的批量處理和標準化
- **特點**：
  - 支持添加單個或多個圖片文件
  - 支持添加整個文件夾中的圖片
  - 支持批量旋轉圖片（-360° 到 360°）
  - 支持批量縮放圖片（1% 到 500%）
  - 支持批量重命名（數字序列或自定義映射）
  - 支持背景填充（可自定義顏色和邊距）
  - 支持吸色器功能（從預覽圖片中選擇顏色）
  - 使用左右方向鍵預覽圖片
  - 可以選擇排除特定圖片不進行批量處理
  - 支持的圖片格式：PNG、JPG、JPEG、BMP、GIF

### 6. 圖片切割工具 (`image_slicer/`)
- **功能**：通過拖動線條來切割圖片
- **用途**：將大圖切割成小塊，用於製作遊戲精靈圖（sprite sheet）
- **特點**：
  - 支持載入多種格式的圖片（PNG、JPG、JPEG、GIF、BMP、TIFF）
  - 可以添加多條垂直和水平切割線
  - 通過拖動可以調整切割線的位置
  - 自動將切割後的圖片保存到指定目錄
  - 保持原始圖片質量

### 7. 圖片旋轉預覽工具 (`imgturn/`)
- **功能**：實時預覽和旋轉圖片
- **用途**：快速預覽和調整遊戲素材的角度
- **特點**：
  - 上傳圖片（支持拖放功能）
  - 實時旋轉預覽
  - 保持圖片透明度
  - 下載旋轉後的圖片
  - 簡潔直觀的界面
  - 支持所有常見的圖片格式（PNG、JPEG、GIF等）

### 8. 坐標排序工具 (`sort_coordinates/`)
- **功能**：處理包含glm::vec2坐標對的文件，將坐標按照從左上角到右下角的順序進行排序
- **用途**：整理遊戲中的坐標數據，確保渲染順序正確
- **特點**：
  - 解析包含`glm::vec2`格式坐標的文件
  - 按照從上到下、從左到右的順序排序坐標
  - 保持原始文件的大括號結構
  - 輸出排序後的結果到新文件或直接修改原文件
  - 自動記錄操作到log.md文件

## 🚀 快速開始

### 安裝依賴

```bash
# 對於 Python 工具
pip install -r requirements.txt

# 或者手動安裝常用依賴
pip install PyQt6 Pillow numpy
```

### 運行工具

#### Python 工具
```bash
# 區域標記工具
cd area_marker
python area_marker_tool.py

# 圖片濾鏡工具
cd color_filter
python color_filter_tool.py

# Windows圖標生成器
cd ico_generator
python ico_generator.py

# 圖片批量處理工具
cd image_processor_main
python image_processor.py

# 圖片切割工具
cd image_slicer
python image_slicer.py

# 坐標排序工具
cd sort_coordinates
python sort_coordinates.py
```

#### Web 工具
```bash
# 圖片裁剪與旋轉工具
cd image_crop_rotate
# 開啟 index.html 文件

# 圖片旋轉預覽工具
cd imgturn
# 開啟 index.html 文件
```

## 🔧 使用說明

### 圖片批量處理工具
1. 點擊"添加文件夾"或"添加圖片文件"按鈕來導入圖片
2. 使用左右方向鍵瀏覽圖片
3. 設置旋轉角度（-360° 到 360°）
4. 設置縮放比例（1% 到 500%）
5. 如果需要背景填充：
   - 勾選"啟用背景填充"選項
   - 設置填充邊距（像素）
   - 使用以下方式之一選擇背景顏色：
     * 點擊"選擇顏色"按鈕從調色板選擇
     * 點擊"吸取顏色"按鈕，然後在預覽圖片上點擊或拖動來選擇顏色（按ESC取消）
6. 如果需要重命名：
   - 勾選"啟用重命名"選項
   - 選擇重命名模式：
     * 數字序列：可以在文本框中輸入前綴（可選），圖片將按照數字順序重命名
     * 映射模式：在映射輸入框中輸入映射規則，格式為 [原名:新名,原名:新名,...]
7. 如果不想處理某張圖片，可以勾選"排除當前圖片"
8. 點擊"處理圖片"按鈕開始批量處理

### 圖片切割工具
1. 運行程序並選擇要切割的圖片
2. 使用「添加垂直線」和「添加水平線」按鈕添加切割線
3. 拖動切割線調整位置
4. 點擊「切割圖片」按鈕，選擇保存目錄
5. 程序會自動將切割後的圖片保存到指定目錄

### 區域標記工具
1. 啟動程序並載入圖片
2. 選擇繪畫狀態（繪製方塊/移動方塊）
3. 從下拉菜單選擇區域類型（"不能放的"或"水路"）
4. 點擊兩個點來定義矩形
5. 使用快捷鍵 Ctrl+Z 撤銷操作
6. 完成標記後導出數據

### 圖片裁剪與旋轉工具（Web）
1. 開啟 index.html 文件
2. 點擊「選擇圖片」按鈕上傳圖片
3. 在圖片上點擊不同位置來選擇區域頂點
4. 點選至少三個點以形成一個封閉區域
5. 使用旋轉滑動條調整圖片角度
6. 點擊「下載圖片」按鈕保存處理後的圖像

## 📝 注意事項

- 處理大量圖片時可能需要等待一段時間
- 建議在處理前備份原始圖片
- 重命名功能支持兩種模式：數字序列模式和映射模式
- 如果未啟用重命名，處理後的圖片將添加"_processed"後綴
- 背景填充會保持原圖的透明度，只在擴展的區域填充所選顏色
- 使用吸色器時，可以在預覽圖片上拖動來即時預覽不同位置的顏色
- 所有操作都會自動記錄到 log.md 文件中

## 🎮 遊戲開發應用

這些工具在 OOP 遊戲開發中的具體應用：

1. **圖片批量處理工具**：用於統一處理遊戲素材的尺寸和格式
2. **圖片切割工具**：將大型精靈圖切割成單個角色或物件圖片
3. **圖片裁剪與旋轉工具**：精確裁剪 UI 元素和遊戲物件
4. **圖片旋轉預覽工具**：快速調整遊戲素材的角度
5. **圖片濾鏡工具**：為遊戲場景添加氛圍效果
6. **區域標記工具**：標記遊戲中的碰撞區域和可互動區域
7. **坐標排序工具**：整理遊戲中的渲染順序和位置數據
8. **Windows圖標生成器**：為遊戲生成應用程序圖標
