# Windows图标生成器

这是一个简单的Python工具，用于将图片转换为Windows图标（.ico）文件。

## 功能特点

- 支持多种常用图片格式转换为ICO
- 自动生成多种尺寸的图标（16x16 到 256x256）
- 保持透明度（如果源图片支持）
- 使用高质量的重采样算法

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：
```bash
python ico_generator.py
```

2. 按提示输入：
   - 源图片路径
   - 输出ICO文件路径（可选，按回车使用默认路径）

## 注意事项

- 建议使用PNG格式的源图片以获得最佳效果
- 源图片最好是正方形的
- 建议源图片尺寸至少为256x256像素
